#  Clínica Saúde & Vida - Site Institucional

Um projeto de site institucional para uma clínica médica fictícia, desenvolvido inteiramente com **HTML5 puro** como parte de um desafio prático de estruturação web.

O objetivo principal deste projeto é consolidar os fundamentos do desenvolvimento web, aplicando marcação semântica e estruturação correta de informações antes da introdução de estilização (CSS) e interatividade (JavaScript).

## Tecnologias Utilizadas

* **HTML5:** Linguagem de marcação utilizada para construir toda a estrutura estrutural das páginas.

##  Funcionalidades e Conceitos Aplicados

O site é composto por 4 páginas interligadas (`index`, `sobre`, `horarios` e `contato`), demonstrando na prática os seguintes conceitos:

* **Tags Semânticas HTML5:** Uso de `<header>`, `<nav>`, `<main>` e `<footer>` para melhor acessibilidade, organização do código e SEO.
* **Navegação Integrada:** Menu de links unindo todas as páginas do sistema.
* **Tabelas de Dados:** Estruturação de grade de horários e serviços médicos utilizando `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>` e `<td>`.
* **Mídias e Iframes:** Inserção de imagens estáticas e incorporação de um mapa interativo utilizando a tag `<iframe>` apontando para o Google Maps.
* **Formulários HTML:** Criação de um formulário de contato funcional estruturalmente, utilizando rótulos (`<label>`), campos de texto (`<input>`, `<textarea>`), validações nativas do navegador (`required`) e tipos específicos de dados (`type="email"`).

##  Estrutura de Arquivos

* `index.html` - Página principal com a apresentação da clínica.
* `sobre.html` - Página detalhando a história, missão e os valores da instituição.
* `horarios.html` - Tabela completa com serviços disponíveis e horários de funcionamento.
* `contato.html` - Informações de contato, mapa embutido e formulário de envio de mensagens.
